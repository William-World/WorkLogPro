# WorkLog Pro

WorkLog Pro is an ASP.NET Core MVC web application for tracking workplace tasks, issues, categories, statuses, priorities, due dates, and follow-ups.

## Features

- Create, view, edit, and delete work items
- Dashboard with work item statistics
- Search and filtering by title, status, and category
- Priority and status tracking
- Follow-up tracking
- Category-based organization
- Bootstrap-styled responsive UI

## Technologies

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- Bootstrap
- ViewModels
- Services
- Dependency Injection
- Async/Await

## Project Purpose

This project was built to practice real-world ASP.NET Core MVC development patterns including CRUD operations, service-layer architecture, EF Core database access, ViewModels, Dependency Injection, and Bootstrap UI design.
